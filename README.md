# 📔 Moogle's Save Book

> "Let me record your tale, kupo! Don't forget to write back to the other Moogles."

This repository serves as a **central vault** for all my gaming data. Instead of hunting through local AppData folders, everything is backed up here for easy syncing and reference.

---

### 📂 What's Inside?
* **💾 Saves:** Progress files for RPGs and long-term campaigns.
* **⚙️ Configs:** Optimized `.ini` and `.cfg` files for peak performance.
* **📜 Guides:** Personal notes, maps, and "to-do" lists for 100% completion.
* **⚔️ Builds:** Character stat spreads and equipment loadouts.
